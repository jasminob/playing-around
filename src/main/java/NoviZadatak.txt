Prosiriti program sa sljedecim:

Napraviti klasu StudentFactory sa jednom jedinom metodom:
  - public static Student createStudent(String ime, String prezime, int index); koja vraca novog studenta sa datim podacima

Napraviti klasu OdsijekFactory analogno kao StudentFactory (metoda createOdsijek sa analognim osobinama kao createStudent)

Rewrite main tako da koristi ove metode i napuni fakultet odsijecima i studentima.

Uraditi dekompoziciju klasa. Dekompozicija podrazumjeva da se svaka klasa nalazi u svom fajlu
(kao sto je klasa Application unutar fajla Application.java treba da bude klasa Student unutar Student.java fajla)

U klasi Predmet dodati polje int minBrojStudenata koji je po defaultu 0 (ne zadaje se kroz konstruktor, nula je)
Ako treba da bude druga vrijednost ona ce se postaviti kroz setter.

U klasi Predmet dodati metodu int brojPotrebnihStudenata() koja vraca broj studenata koji se jos moraju upisati u ovaj
Predmet da bi se mogao drzati.

U klasi Odsijek takodjer dodati metodu int brojPotrebnihStudenata(int godina) koja vraca broj studenata koji se jos moraju
upisati u ovaj Odsijek na datu godinu da bi se svi predmeti mogli realizovati (ona radi slicno kao metoda maxBrojStudenata)
